# Contributing

Please follow the contributor workflow and validation checklist in [docs/contributing.md](docs/contributing.md).

Quick checklist before opening a PR:

- Run pre-flight checks (`pnpm`, `bun`, `rustc`, `cargo`)
- Build and test affected packages
- Update `MODULE.md`, `TODO.md`, and `PROJECT_PLAN.md` for impacted areas
- Ensure no secrets or runtime-generated artifacts are committed
