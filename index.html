Arkestrator Linux Packages

Install Arkestrator on your Linux system using your package manager.

Quick Install (auto-detects distro)

curl -fsSL https://timvanhelsdingen.github.io/arkestrator/install.sh | sudo bash

Fedora / RHEL / CentOS (dnf/yum)

sudo dnf config-manager addrepo --from-repofile=https://timvanhelsdingen.github.io/arkestrator/arkestrator.repo
sudo dnf install arkestrator

Debian / Ubuntu (apt)

curl -fsSL https://timvanhelsdingen.github.io/arkestrator/gpg-key.asc | sudo gpg --dearmor -o /usr/share/keyrings/arkestrator.gpg
echo "deb [signed-by=/usr/share/keyrings/arkestrator.gpg] https://timvanhelsdingen.github.io/arkestrator/apt stable main" | sudo tee /etc/apt/sources.list.d/arkestrator.list
sudo apt update && sudo apt install arkestrator

Arch Linux (AUR)

yay -S arkestrator-bin

Or manually: git clone https://aur.archlinux.org/arkestrator-bin.git && cd arkestrator-bin && makepkg -si
