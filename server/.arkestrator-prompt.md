## Game Concept
- Player tilts the LEVEL using arrow keys/WASD — the ball itself has NO direct controls
- A ball (RigidBody3D) rolls due to gravity on the tilted level
- 5 levels of increasing difficulty with countdown timers
- Fall off the edge = restart the level
- Reach the goal zone = level complete, record time as score
- Highscore system tracks best completion times per level
- Full game flow: Main Menu → Level Select → Play Level → Win/Lose → Level Select

### highscore_manager.gd (AUTOLOAD as "HighscoreManager")

### level_tilter.gd (attached to "LevelTilter" Node3D in each level)
```
- @export var max_tilt_angle: float = 15.0  # degrees
- @export var tilt_speed: float = 3.0
- var target_rotation: Vector2 = Vector2.ZERO
- func _physics_process(delta):
    var input = Vector2.ZERO
    input.x = Input.get_axis("tilt_left", "tilt_right")    # Z-axis tilt
    input.y = Input.get_axis("tilt_forward", "tilt_backward")  # X-axis tilt
    target_rotation = input * deg_to_rad(max_tilt_angle)
    rotation.x = lerp(rotation.x, target_rotation.y, tilt_speed * delta)
    rotation.z = lerp(rotation.z, -target_rotation.x, tilt_speed * delta)
```

### ball_controller.gd (attached to RigidBody3D)
```
- var start_position: Vector3
- func _ready(): start_position = global_position
- func reset_position(): global_position = start_position, linear_velocity = Vector3.ZERO, angular_velocity = Vector3.ZERO
- NO _physics_process movement code — ball moves only from gravity + level tilt
```

### camera_controller.gd (attached to Camera3D)
```
- @export var target: NodePath
- @export var offset: Vector3 = Vector3(0, 10, 8)
- @export var smooth_speed: float = 5.0
- var ball: Node3D
- func _ready(): ball = get_node(target) if target else null
- func _physics_process(delta):
    if ball:
        var desired = ball.global_position + offset
        global_position = global_position.lerp(desired, smooth_speed * delta)
        look_at(ball.global_position, Vector3.UP)
```

### goal_zone.gd (attached to Area3D)
```
- signal goal_reached
- func _ready(): body_entered.connect(_on_body_entered), set collision mask to detect ball (layer 2)
- func _on_body_entered(body): if body.is_in_group("ball"): goal_reached.emit()
```

### fall_zone.gd (attached to Area3D)
```
- signal ball_fell
- func _ready(): body_entered.connect(_on_body_entered), set collision mask layer 2
- func _on_body_entered(body): if body.is_in_group("ball"): ball_fell.emit()
```

### game_scene.gd (attached to Game root):
```
- var current_level_instance: Node
- var time_remaining: float
- var is_playing: bool = false
- func _ready(): load level from GameManager.current_level, setup timer
- func _physics_process(delta): countdown timer, check time up
- func load_level(level_num):
    - Remove old level from LevelContainer
    - Instance new level scene, add to LevelContainer
    - Position ball at level's StartPosition marker
    - Connect goal_zone.goal_reached and fall_zone.ball_fell
    - Set time_remaining = GameManager.level_time_limits[level_num-1]
    - is_playing = true
- func _on_goal_reached(): is_playing=false, calculate elapsed time, GameManager.complete_level()
- func _on_ball_fell(): brief pause, reset ball to start position (don't restart timer)
- func _on_time_up(): is_playing=false, GameManager.fail_level()
- func _input(event): if R pressed: reset ball position; if Escape: GameManager.go_to_menu()
```
